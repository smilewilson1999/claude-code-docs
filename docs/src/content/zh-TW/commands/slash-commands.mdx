# 斜槓命令

斜槓命令提供對互動式 REPL 中功能的快速訪問。

## 使用方法

輸入 `/` 加命令名：

```
> /commit
```

命令可以接受引數：

```
> /diff --staged
```

## 自動補全

REPL 為斜槓命令提供 Tab 補全。輸入 `/` 並按 Tab 檢視可用命令。

## 與技能系統的關係

一些斜槓命令由[技能系統](/zh-TW/subsystems/skills)驅動。技能是擴充套件的命令定義，可以：

- 載入自定義提示詞
- 連結多個操作
- 接受結構化輸入
- 提供專門的 UI

當使用者透過 `/skill-name` 呼叫技能時，技能的提示詞會被展開並執行。

## 常用斜槓命令

| 命令 | 分類 | 描述 |
|------|------|------|
| `/help` | 資訊 | 列出可用命令 |
| `/clear` | 會話 | 清除對話 |
| `/compact` | 會話 | 壓縮上下文 |
| `/commit` | Git | 建立 AI 輔助提交 |
| `/diff` | Git | 顯示變更 |
| `/cost` | 資訊 | Token/費用摘要 |
| `/config` | 配置 | 編輯設定 |
| `/mcp` | 擴充套件 | 管理 MCP 伺服器 |

## 解析

斜槓命令由 `src/commands.ts` 中的命令系統解析。解析器：

1. 檢測使用者輸入中的 `/` 字首
2. 提取命令名和引數
3. 解析到已註冊的命令處理器
4. 使用解析後的引數執行
