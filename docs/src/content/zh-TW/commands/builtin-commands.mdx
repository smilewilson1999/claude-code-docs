# 內建命令

Claude Code 中所有 103+ 內建命令的參考。每個命令作為 `src/commands/` 下的模組實現。

## Git 與版本控制

| 命令 | 描述 |
|------|------|
| `/commit` | 使用 AI 生成的訊息建立 git 提交 |
| `/diff` | 顯示當前 git diff |
| `/commit-push-pr` | 提交、推送並建立 Pull Request |
