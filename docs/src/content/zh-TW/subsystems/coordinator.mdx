# 協調器

**原始碼**: `src/coordinator/` — `coordinatorMode.ts`

## 概述

協調器是一個多代理編排系統，支援跨多個 Claude 例項的並行任務執行。此功能透過 `COORDINATOR_MODE` 特性標誌門控。

## 工作原理

協調器作為監督代理：

1. 分析複雜任務並拆分為子任務
2. 將子任務分配給專門的代理
3. 監控所有代理的進度
4. 將結果聚合為連貫的輸出

## 與 Agent 工具的關係

[Agent 工具](/zh-TW/tools/agent-tool)支援一次性的子代理生成，而協調器提供更高階別的編排模式：

| 特性 | Agent 工具 | 協調器 |
|------|-----------|--------|
| 範圍 | 單次委派 | 多代理編排 |
| 規劃 | 呼叫者決定 | 協調器規劃 |
| 並行性 | 手動 | 自動 |
| 監控 | 按代理 | 統一儀表板 |
