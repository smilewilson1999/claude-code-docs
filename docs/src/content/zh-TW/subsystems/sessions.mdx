# 會話管理

**原始碼**: `src/history.ts` (464 行)

## 概述

Claude Code 維護會話歷史，支援恢復對話、匯出記錄和管理多個會話。

## 會話儲存

會話儲存在本地，包含：

- 對話訊息（使用者、助手、系統）
- 工具呼叫歷史和結果
- 時間戳和後設資料
- Token 使用統計

## 恢復

恢復之前的會話：

```bash
claude --resume
```

這將恢復完整的對話上下文，包括工具結果和 AI 狀態。

## 會話命令

| 命令 | 描述 |
|------|------|
| `/session` | 會話管理工具 |
| `/resume` | 恢復特定會話 |
| `/clear` | 開始新會話 |
| `/compact` | 壓縮當前會話上下文 |

## 上下文壓縮

`/compact` 命令壓縮對話歷史以釋放上下文視窗空間，同時保留關鍵資訊。適用於接近上下文限制的長會話。
