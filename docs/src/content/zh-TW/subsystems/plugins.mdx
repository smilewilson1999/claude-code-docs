# 外掛系統

**原始碼**: `src/plugins/` — `builtinPlugins.ts`、`bundled/`

## 概述

外掛系統提供了一個擴充套件點，用於在工具和技能之外為 Claude Code 新增功能。外掛可以鉤入應用生命週期並修改行為。

## 架構

外掛透過 `builtinPlugins.ts` 註冊，在應用初始化時載入。每個外掛可以：

- 註冊新工具
- 新增事件監聽器
- 修改渲染管線
- 擴充套件命令系統

## 內建外掛

內建外掛隨 Claude Code 一起釋出在 `src/plugins/bundled/` 中，為模組化提供核心功能實現。

## 與 MCP 的關係

MCP 伺服器透過標準化協議提供外部工具整合，而外掛在 Claude Code 程序內部執行。與 MCP 伺服器相比，外掛對應用內部有更深層的訪問許可權。
