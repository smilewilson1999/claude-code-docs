# 子系統

擴充套件 Claude Code 核心能力的主要子系統。

- [**MCP 整合**](/zh-TW/subsystems/mcp) — 用於外部工具伺服器的 Model Context Protocol
- [**技能系統**](/zh-TW/subsystems/skills) — 內建和自定義技能定義
- [**外掛系統**](/zh-TW/subsystems/plugins) — 第三方外掛架構
- [**遠端橋接**](/zh-TW/subsystems/bridge) — 遠端會話轉發和執行
- [**Vim 模式**](/zh-TW/subsystems/vim) — Vim 鍵繫結和模態編輯
- [**協調器**](/zh-TW/subsystems/coordinator) — 多代理任務協調
- [**會話管理**](/zh-TW/subsystems/sessions) — 會話持久化、恢復和匯出
