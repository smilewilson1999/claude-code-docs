# Vim 模式

**原始碼**: `src/vim/`（6 個檔案）

## 概述

Claude Code 支援輸入區域的 vim 風格模態編輯。啟用後，使用者可以使用熟悉的 vim 移動、運算子和文字物件。

## 啟用

透過 `/vim` 命令切換 Vim 模式。啟用後：

- **Normal 模式** — 導航和執行命令
- **Insert 模式** — 輸入文字（透過 `i`、`a`、`o` 等進入）
- **Visual 模式** — 選擇文字進行操作

## 支援的功能

### 移動
- 字元：`h`、`l`
- 單詞：`w`、`b`、`e`、`W`、`B`、`E`
- 行：`0`、`$`、`^`
- 搜尋：`f`、`F`、`t`、`T`

### 運算子
- 刪除：`d`、`dd`、`D`
- 修改：`c`、`cc`、`C`
- 複製：`y`、`yy`、`Y`

### 文字物件
- 單詞：`iw`、`aw`
- 引號：`i"`、`a"`、`i'`、`a'`
- 括號：`i(`、`a(`、`i{`、`a{`
