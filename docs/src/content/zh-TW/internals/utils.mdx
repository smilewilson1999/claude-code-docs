# 工具函式

**原始碼**: `src/utils/`（298+ 檔案）

## 概述

Claude Code 擁有廣泛的工具函式庫，涵蓋 git 操作、shell 互動、配置管理、認證、檔案操作等。

## 主要分類

### Git 工具函式
倉庫分析和操作：倉庫檢測、分支管理、diff 生成、提交歷史。

### Bash 工具函式
Shell 互動輔助：命令執行、輸出解析、環境檢測、Shell 初始化。

### 配置工具函式
設定管理：配置檔案讀寫、設定遷移、預設值管理、驗證。

### 認證工具函式
認證支援：令牌管理、憑證儲存、提供商檢測。

### 檔案工具函式
檔案系統操作：路徑解析、檔案型別檢測、內容雜湊、臨時檔案管理。

### 分析工具函式
程式碼分析輔助：語言檢測、專案型別推斷、依賴分析。

## 使用模式

工具函式由需要它們的模組直接匯入：

```typescript
import { getGitStatus } from "@/utils/git";
import { parseConfig } from "@/utils/config";
```
