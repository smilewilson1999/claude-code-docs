# 鍵繫結

**原始碼**: `src/keybindings/`（16 個檔案）

## 概述

Claude Code 擁有全面的鍵繫結系統，支援預設快捷鍵、自定義覆蓋和和絃序列。

## 配置

自定義鍵繫結定義在 `~/.claude/keybindings.json` 中：

```json
[
  {
    "key": "ctrl+s",
    "command": "submit"
  },
  {
    "key": "ctrl+k ctrl+c",
    "command": "clear"
  }
]
```

## 預設鍵繫結

| 按鍵 | 操作 |
|------|------|
| `Enter` | 提交輸入 |
| `Ctrl+C` | 取消/中斷 |
| `Ctrl+D` | 退出 |
| `Up/Down` | 導航歷史 |
| `Tab` | 自動補全 |
| `Escape` | 取消當前操作 |

## 和絃序列

鍵繫結支援和絃序列（多鍵組合）：

```
ctrl+k ctrl+c  — 兩步和絃
```

鍵繫結系統會等待完整的和絃完成後再執行命令。
