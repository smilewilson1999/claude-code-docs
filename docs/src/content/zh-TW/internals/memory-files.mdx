# 記憶檔案

**原始碼**: `src/memdir/`

## 概述

Claude Code 使用記憶檔案系統在會話間持久化資訊。這使 AI 能夠隨時間積累關於專案和使用者偏好的知識。

## CLAUDE.md 約定

`CLAUDE.md` 檔案是專案級指令檔案，Claude Code 會自動將其載入到上下文中：

```markdown
# 專案指令

- 使用 TypeScript strict 模式
- 使用 `bun test` 執行測試
- 遵循現有程式碼模式
```

### 載入順序

CLAUDE.md 檔案從多個位置載入（按順序）：

1. `~/.claude/CLAUDE.md` — 全域性指令
2. 父目錄（從專案根向上遍歷）
3. `./CLAUDE.md` — 專案根指令

## 自動記憶目錄

自動記憶系統將持久筆記儲存在：

```
~/.claude/projects/<project-hash>/memory/
└── MEMORY.md          # 主記憶檔案
```

### 工作原理

- `MEMORY.md` 自動載入到對話上下文中
- AI 可以寫入觀察、模式和偏好
- 內容在會話間持久儲存
- 檔案按主題語義組織

### 儲存內容

- 跨多次互動確認的穩定模式
- 架構決策和重要檔案路徑
- 使用者對工作流和工具的偏好
- 反覆出現問題的解決方案

## 記憶管理

`/memory` 命令提供：

- 檢視當前記憶檔案
- 編輯記憶內容
- 清除特定記憶
- 管理按專案的記憶
