# 常量配置

**原始碼**: `src/constants/`（23+ 子目錄）

## 概述

constants 目錄包含 Claude Code 中使用的配置值、提示模板、模型定義和主題規格。

## 系統提示詞

系統提示詞定義 Claude Code 的行為和指令，包括基礎性格和能力、工具使用說明、安全指南和輸出格式規則。

## 模型配置

模型定義指定：

| 引數 | 描述 |
|------|------|
| Model ID | Claude 模型識別符號 |
| Context window | 最大 token 容量 |
| Max output | 最大響應 token 數 |
| Capabilities | 支援的功能（視覺、工具等） |

## 主題定義

`src/constants/themes/` 中的主題檔案定義顏色方案（淺色/深色）、語法高亮顏色、UI 元素顏色和狀態指示器顏色。

## 提示模板

可重用的提示模板，用於提交訊息生成、程式碼審查、錯誤解釋和重構建議。

## 組織結構

常量按領域組織：

```
src/constants/
├── prompts/        # 系統和使用者提示詞
├── models/         # 模型定義
├── themes/         # 顏色主題
└── ...             # 其他配置
```
