# 服務層

**原始碼**: `src/services/`（25+ 子目錄）

服務層提供 API 通訊、分析、程式碼智慧和認證的基礎設施。

- [**API 服務**](/zh-TW/services/api) — Claude API 整合，支援流式傳輸和重試
- [**分析與遙測**](/zh-TW/services/analytics) — 遙測和使用分析
- [**LSP 整合**](/zh-TW/services/lsp) — 語言伺服器協議，提供程式碼智慧
- [**OAuth**](/zh-TW/services/oauth) — 基於瀏覽器的認證流程
