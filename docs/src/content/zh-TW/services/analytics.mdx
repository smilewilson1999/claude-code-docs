# 分析與遙測

**原始碼**: `src/services/analytics/`

## 概述

Claude Code 收集尊重隱私的使用分析以改進產品。分析服務整合了 GrowthBook 用於特性標誌評估和 A/B 測試。

## 元件

### 遙測
匿名使用指標，包括會話時長、工具使用頻率、錯誤率、功能採用率。

### GrowthBook 整合
提供特性標誌評估、A/B 測試分配和漸進發布管理。

## 隱私

所有分析均為：
- 匿名 — 無個人身份資訊
- 可退出 — 使用者可禁用遙測
- 聚合 — 不儲存個人資料
- 最小化 — 僅收集必要指標
