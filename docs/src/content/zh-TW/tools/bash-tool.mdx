# Bash 工具

**原始碼**: `src/tools/BashTool/`

## 概述

Bash 工具在使用者環境中執行 shell 命令。它是最常用的工具之一，使 Claude Code 能夠執行構建、測試、git 操作和任意 shell 命令。

## 引數

- **command** — 要執行的 shell 命令
- **description** — 命令的人類可讀描述
- **timeout** — 可選超時（毫秒，最大 600,000ms / 10 分鐘）
- **run_in_background** — 非同步執行，不阻塞

## 執行模型

命令在持久的 shell 會話中執行：

- 工作目錄在命令間持久儲存
- Shell 狀態（環境變數、別名）不在呼叫間持久儲存
- Shell 從使用者的 profile（bash 或 zsh）初始化

## 安全性

Bash 工具實現了多重安全措施：

- **許可權門控** — 使用者必須批准命令執行
- **沙箱** — 命令在配置的沙箱邊界內執行
- **超時強制** — 防止失控程序
- **描述要求** — AI 必須描述命令的功能

## 後臺執行

使用 `run_in_background: true` 時：

- 命令非同步執行
- 完成時通知使用者
- 輸出可透過 `TaskOutput` 獲取
- 適用於長時間執行的程序（開發伺服器、構建）

## 進度流

命令輸出透過 `BashProgress` 事件實時流式傳輸，使用者可以在輸出產生時看到。
