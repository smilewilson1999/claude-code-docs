一個執行在終端中的 AI 程式設計助手，基於 TypeScript 和 React (Ink) 構建。

## 技術棧

- **執行時**: Bun / Node.js
- **語言**: TypeScript (strict 模式)
- **UI 框架**: 基於自定義 Ink 引擎的 React
- **AI 後端**: Claude API (流式傳輸)
- **擴充套件協議**: Model Context Protocol (MCP)
- **構建**: Bun bundler，支援特性標誌死程式碼消除
