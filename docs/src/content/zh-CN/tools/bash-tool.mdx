# Bash 工具

**源码**: `src/tools/BashTool/`

## 概述

Bash 工具在用户环境中执行 shell 命令。它是最常用的工具之一，使 Claude Code 能够运行构建、测试、git 操作和任意 shell 命令。

## 参数

- **command** — 要执行的 shell 命令
- **description** — 命令的人类可读描述
- **timeout** — 可选超时（毫秒，最大 600,000ms / 10 分钟）
- **run_in_background** — 异步运行，不阻塞

## 执行模型

命令在持久的 shell 会话中执行：

- 工作目录在命令间持久保存
- Shell 状态（环境变量、别名）不在调用间持久保存
- Shell 从用户的 profile（bash 或 zsh）初始化

## 安全性

Bash 工具实现了多重安全措施：

- **权限门控** — 用户必须批准命令执行
- **沙箱** — 命令在配置的沙箱边界内运行
- **超时强制** — 防止失控进程
- **描述要求** — AI 必须描述命令的功能

## 后台执行

使用 `run_in_background: true` 时：

- 命令异步运行
- 完成时通知用户
- 输出可通过 `TaskOutput` 获取
- 适用于长时间运行的进程（开发服务器、构建）

## 进度流

命令输出通过 `BashProgress` 事件实时流式传输，用户可以在输出产生时看到。
