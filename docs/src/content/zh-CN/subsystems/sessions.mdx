# 会话管理

**源码**: `src/history.ts` (464 行)

## 概述

Claude Code 维护会话历史，支持恢复对话、导出记录和管理多个会话。

## 会话存储

会话存储在本地，包含：

- 对话消息（用户、助手、系统）
- 工具调用历史和结果
- 时间戳和元数据
- Token 使用统计

## 恢复

恢复之前的会话：

```bash
claude --resume
```

这将恢复完整的对话上下文，包括工具结果和 AI 状态。

## 会话命令

| 命令 | 描述 |
|------|------|
| `/session` | 会话管理工具 |
| `/resume` | 恢复特定会话 |
| `/clear` | 开始新会话 |
| `/compact` | 压缩当前会话上下文 |

## 上下文压缩

`/compact` 命令压缩对话历史以释放上下文窗口空间，同时保留关键信息。适用于接近上下文限制的长会话。
