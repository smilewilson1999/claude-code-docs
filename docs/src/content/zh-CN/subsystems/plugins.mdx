# 插件系统

**源码**: `src/plugins/` — `builtinPlugins.ts`、`bundled/`

## 概述

插件系统提供了一个扩展点，用于在工具和技能之外为 Claude Code 添加功能。插件可以钩入应用生命周期并修改行为。

## 架构

插件通过 `builtinPlugins.ts` 注册，在应用初始化时加载。每个插件可以：

- 注册新工具
- 添加事件监听器
- 修改渲染管线
- 扩展命令系统

## 内置插件

内置插件随 Claude Code 一起发布在 `src/plugins/bundled/` 中，为模块化提供核心功能实现。

## 与 MCP 的关系

MCP 服务器通过标准化协议提供外部工具集成，而插件在 Claude Code 进程内部运行。与 MCP 服务器相比，插件对应用内部有更深层的访问权限。
