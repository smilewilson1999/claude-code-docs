# Vim 模式

**源码**: `src/vim/`（6 个文件）

## 概述

Claude Code 支持输入区域的 vim 风格模态编辑。启用后，用户可以使用熟悉的 vim 移动、操作符和文本对象。

## 激活

通过 `/vim` 命令切换 Vim 模式。激活后：

- **Normal 模式** — 导航和执行命令
- **Insert 模式** — 输入文本（通过 `i`、`a`、`o` 等进入）
- **Visual 模式** — 选择文本进行操作

## 支持的功能

### 移动
- 字符：`h`、`l`
- 单词：`w`、`b`、`e`、`W`、`B`、`E`
- 行：`0`、`$`、`^`
- 搜索：`f`、`F`、`t`、`T`

### 操作符
- 删除：`d`、`dd`、`D`
- 修改：`c`、`cc`、`C`
- 复制：`y`、`yy`、`Y`

### 文本对象
- 单词：`iw`、`aw`
- 引号：`i"`、`a"`、`i'`、`a'`
- 括号：`i(`、`a(`、`i{`、`a{`
