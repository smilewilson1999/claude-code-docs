# 子系统

扩展 Claude Code 核心能力的主要子系统。

- [**MCP 集成**](/zh-CN/subsystems/mcp) — 用于外部工具服务器的 Model Context Protocol
- [**技能系统**](/zh-CN/subsystems/skills) — 内置和自定义技能定义
- [**插件系统**](/zh-CN/subsystems/plugins) — 第三方插件架构
- [**远程桥接**](/zh-CN/subsystems/bridge) — 远程会话转发和执行
- [**Vim 模式**](/zh-CN/subsystems/vim) — Vim 键绑定和模态编辑
- [**协调器**](/zh-CN/subsystems/coordinator) — 多代理任务协调
- [**会话管理**](/zh-CN/subsystems/sessions) — 会话持久化、恢复和导出
