# 协调器

**源码**: `src/coordinator/` — `coordinatorMode.ts`

## 概述

协调器是一个多代理编排系统，支持跨多个 Claude 实例的并行任务执行。此功能通过 `COORDINATOR_MODE` 特性标志门控。

## 工作原理

协调器作为监督代理：

1. 分析复杂任务并拆分为子任务
2. 将子任务分配给专门的代理
3. 监控所有代理的进度
4. 将结果聚合为连贯的输出

## 与 Agent 工具的关系

[Agent 工具](/zh-CN/tools/agent-tool)支持一次性的子代理生成，而协调器提供更高级别的编排模式：

| 特性 | Agent 工具 | 协调器 |
|------|-----------|--------|
| 范围 | 单次委派 | 多代理编排 |
| 规划 | 调用者决定 | 协调器规划 |
| 并行性 | 手动 | 自动 |
| 监控 | 按代理 | 统一仪表板 |
