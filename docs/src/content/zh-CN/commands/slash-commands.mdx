# 斜杠命令

斜杠命令提供对交互式 REPL 中功能的快速访问。

## 使用方法

输入 `/` 加命令名：

```
> /commit
```

命令可以接受参数：

```
> /diff --staged
```

## 自动补全

REPL 为斜杠命令提供 Tab 补全。输入 `/` 并按 Tab 查看可用命令。

## 与技能系统的关系

一些斜杠命令由[技能系统](/zh-CN/subsystems/skills)驱动。技能是扩展的命令定义，可以：

- 加载自定义提示词
- 链接多个操作
- 接受结构化输入
- 提供专门的 UI

当用户通过 `/skill-name` 调用技能时，技能的提示词会被展开并执行。

## 常用斜杠命令

| 命令 | 分类 | 描述 |
|------|------|------|
| `/help` | 信息 | 列出可用命令 |
| `/clear` | 会话 | 清除对话 |
| `/compact` | 会话 | 压缩上下文 |
| `/commit` | Git | 创建 AI 辅助提交 |
| `/diff` | Git | 显示变更 |
| `/cost` | 信息 | Token/费用摘要 |
| `/config` | 配置 | 编辑设置 |
| `/mcp` | 扩展 | 管理 MCP 服务器 |

## 解析

斜杠命令由 `src/commands.ts` 中的命令系统解析。解析器：

1. 检测用户输入中的 `/` 前缀
2. 提取命令名和参数
3. 解析到已注册的命令处理器
4. 使用解析后的参数执行
