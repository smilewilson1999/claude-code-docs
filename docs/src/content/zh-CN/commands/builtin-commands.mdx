# 内置命令

Claude Code 中所有 103+ 内置命令的参考。每个命令作为 `src/commands/` 下的模块实现。

## Git 与版本控制

| 命令 | 描述 |
|------|------|
| `/commit` | 使用 AI 生成的消息创建 git 提交 |
| `/diff` | 显示当前 git diff |
| `/commit-push-pr` | 提交、推送并创建 Pull Request |
