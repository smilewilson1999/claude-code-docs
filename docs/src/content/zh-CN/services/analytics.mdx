# 分析与遥测

**源码**: `src/services/analytics/`

## 概述

Claude Code 收集尊重隐私的使用分析以改进产品。分析服务集成了 GrowthBook 用于特性标志评估和 A/B 测试。

## 组件

### 遥测
匿名使用指标，包括会话时长、工具使用频率、错误率、功能采用率。

### GrowthBook 集成
提供特性标志评估、A/B 测试分配和渐进发布管理。

## 隐私

所有分析均为：
- 匿名 — 无个人身份信息
- 可退出 — 用户可禁用遥测
- 聚合 — 不存储个人数据
- 最小化 — 仅收集必要指标
