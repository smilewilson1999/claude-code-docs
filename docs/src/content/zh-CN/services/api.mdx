# API 服务

**源码**: `src/services/api/`

## 概述

API 服务管理与 Claude AI 后端的所有通信。处理请求构建、流式响应、错误恢复和速率限制。

## 特性

### 流式传输
所有 AI 响应逐 token 流式传输。API 服务处理来自 Claude API 的 Server-Sent Events (SSE) 以实现实时输出。

### 多提供商支持

| 提供商 | 环境变量 |
|--------|---------|
| Anthropic API | `ANTHROPIC_API_KEY` |
| AWS Bedrock | `CLAUDE_CODE_USE_BEDROCK` |
| Google Vertex | `CLAUDE_CODE_USE_VERTEX` |

### 速率限制
`rateLimiting.ts` 实现了指数退避和重试逻辑，优雅地处理 API 速率限制。

### 错误处理
健壮的错误处理，覆盖网络故障、认证错误、速率限制响应、格式错误的响应和超时条件。
