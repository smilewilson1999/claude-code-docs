## 技术栈

- **运行时**: Bun / Node.js
- **语言**: TypeScript (strict 模式)
- **UI 框架**: 基于自定义 Ink 引擎的 React
- **AI 后端**: Claude API (流式传输)
- **扩展协议**: Model Context Protocol (MCP)
- **构建**: Bun bundler，支持特性标志死代码消除
