# 工具函数

**源码**: `src/utils/`（298+ 文件）

## 概述

Claude Code 拥有广泛的工具函数库，涵盖 git 操作、shell 交互、配置管理、认证、文件操作等。

## 主要分类

### Git 工具函数
仓库分析和操作：仓库检测、分支管理、diff 生成、提交历史。

### Bash 工具函数
Shell 交互辅助：命令执行、输出解析、环境检测、Shell 初始化。

### 配置工具函数
设置管理：配置文件读写、设置迁移、默认值管理、验证。

### 认证工具函数
认证支持：令牌管理、凭证存储、提供商检测。

### 文件工具函数
文件系统操作：路径解析、文件类型检测、内容哈希、临时文件管理。

### 分析工具函数
代码分析辅助：语言检测、项目类型推断、依赖分析。

## 使用模式

工具函数由需要它们的模块直接导入：

```typescript
import { getGitStatus } from "@/utils/git";
import { parseConfig } from "@/utils/config";
```
