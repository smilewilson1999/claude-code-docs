# 记忆文件

**源码**: `src/memdir/`

## 概述

Claude Code 使用记忆文件系统在会话间持久化信息。这使 AI 能够随时间积累关于项目和用户偏好的知识。

## CLAUDE.md 约定

`CLAUDE.md` 文件是项目级指令文件，Claude Code 会自动将其加载到上下文中：

```markdown
# 项目指令

- 使用 TypeScript strict 模式
- 使用 `bun test` 运行测试
- 遵循现有代码模式
```

### 加载顺序

CLAUDE.md 文件从多个位置加载（按顺序）：

1. `~/.claude/CLAUDE.md` — 全局指令
2. 父目录（从项目根向上遍历）
3. `./CLAUDE.md` — 项目根指令

## 自动记忆目录

自动记忆系统将持久笔记存储在：

```
~/.claude/projects/<project-hash>/memory/
└── MEMORY.md          # 主记忆文件
```

### 工作原理

- `MEMORY.md` 自动加载到对话上下文中
- AI 可以写入观察、模式和偏好
- 内容在会话间持久保存
- 文件按主题语义组织

### 保存内容

- 跨多次交互确认的稳定模式
- 架构决策和重要文件路径
- 用户对工作流和工具的偏好
- 反复出现问题的解决方案

## 记忆管理

`/memory` 命令提供：

- 查看当前记忆文件
- 编辑记忆内容
- 清除特定记忆
- 管理按项目的记忆
