# 常量配置

**源码**: `src/constants/`（23+ 子目录）

## 概述

constants 目录包含 Claude Code 中使用的配置值、提示模板、模型定义和主题规格。

## 系统提示词

系统提示词定义 Claude Code 的行为和指令，包括基础性格和能力、工具使用说明、安全指南和输出格式规则。

## 模型配置

模型定义指定：

| 参数 | 描述 |
|------|------|
| Model ID | Claude 模型标识符 |
| Context window | 最大 token 容量 |
| Max output | 最大响应 token 数 |
| Capabilities | 支持的功能（视觉、工具等） |

## 主题定义

`src/constants/themes/` 中的主题文件定义颜色方案（浅色/深色）、语法高亮颜色、UI 元素颜色和状态指示器颜色。

## 提示模板

可重用的提示模板，用于提交消息生成、代码审查、错误解释和重构建议。

## 组织结构

常量按领域组织：

```
src/constants/
├── prompts/        # 系统和用户提示词
├── models/         # 模型定义
├── themes/         # 颜色主题
└── ...             # 其他配置
```
