# 键绑定

**源码**: `src/keybindings/`（16 个文件）

## 概述

Claude Code 拥有全面的键绑定系统，支持默认快捷键、自定义覆盖和和弦序列。

## 配置

自定义键绑定定义在 `~/.claude/keybindings.json` 中：

```json
[
  {
    "key": "ctrl+s",
    "command": "submit"
  },
  {
    "key": "ctrl+k ctrl+c",
    "command": "clear"
  }
]
```

## 默认键绑定

| 按键 | 操作 |
|------|------|
| `Enter` | 提交输入 |
| `Ctrl+C` | 取消/中断 |
| `Ctrl+D` | 退出 |
| `Up/Down` | 导航历史 |
| `Tab` | 自动补全 |
| `Escape` | 取消当前操作 |

## 和弦序列

键绑定支持和弦序列（多键组合）：

```
ctrl+k ctrl+c  — 两步和弦
```

键绑定系统会等待完整的和弦完成后再执行命令。
