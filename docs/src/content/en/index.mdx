## Tech Stack

- **Runtime**: Bun / Node.js
- **Language**: TypeScript (strict)
- **UI Framework**: React via custom Ink engine
- **AI Backend**: Claude API (streaming)
- **Extension Protocol**: Model Context Protocol (MCP)
- **Build**: Bun bundler with feature flag dead-code elimination
