# Subsystems

Major subsystems that extend Claude Code's core capabilities.

- [**MCP Integration**](/en/subsystems/mcp) — Model Context Protocol for external tool servers
- [**Skills System**](/en/subsystems/skills) — Bundled and custom skill definitions
- [**Plugin System**](/en/subsystems/plugins) — Third-party plugin architecture
- [**Remote Bridge**](/en/subsystems/bridge) — Remote session forwarding and execution
- [**Vim Mode**](/en/subsystems/vim) — Vim keybindings and modal editing
- [**Coordinator**](/en/subsystems/coordinator) — Multi-agent task coordination
- [**Session Management**](/en/subsystems/sessions) — Session persistence, resume, and export
